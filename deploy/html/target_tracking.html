Target tracking

Single target detection and target tracking can be applied to a whole layer or a defined region or specific selection in the echogram. Regardless what part of your data you wish to apply the target algorithm to, there are common steps which are outlined next. For this tutorial single target detection and target tracking will be applied to a defined region.

1. First, let's define a region. Press keyboard short-cut 4 or go to the toolbar and selection the region tool, and select from the drop-down menu the Rectangular region. Define a region in the echogram by left-clicking and holding while dragging the mouse to reach the extent of the region you wish to include:



2. In the Single Targets tab of the Algorithms panel define the algorithm parameter you want to apply for single target detection. Next right click on your region and choose “Algorithms", then "Apply Single Target detection”:



Note: Single target detection can be applied to a whole layer or to specific selection following the same steps.

3. Your defined region should now contained detected single targets under the current settings. Remember to adjust the color scale so that there is no “hidden” targets:



4. In the Target tracking tab of the Algorithms panel, define the algorithm parameter you want to use for target tracking, right-click on your region and choose “Algorithms", then "Apply Target Tracking”:



Note: Tracked targets are just shown for demonstration purposes only, and they do not represent real fish tracks. Settings on the Target tracking algorithm tab were chosen for demonstration and they should not necessarily be used for tracking real targets.

5. The last step is to go to the Export menu on the main menu bar, and choose Export Tracked target to xls files. Same as for single targets. Values for each target contained in each track are saved in the xls files, with a Track_ID to be able to sort them track per track.
