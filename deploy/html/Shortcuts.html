Keyboard shortcuts

Keyboard shortcuts for the main display:

Data display:

F: Change frequency shown in the echogram.

E: Change data type shown in the echogram (Power, Angle across, Angle along, Sv, TS Unc).

+: Shift color map up.

-: Shift color map down.

B: Toggle "Display/hide Under Bottom data".

R: Toggle "Display/hide Regions" (filled or boundary only).

T: Toggle "Display/hide bad transmits".


Zoom in/out or move around the display:

Mouse scroll: Zoom in/out.

Arrow Keys: Move up/down/left/right.

W: Zoom in.

S: Zoom out.

A: Move left.

D: Move right.

Important shortcuts:

L: Display/edit logbook.

shift+L: Open multiple logbooks.

N: Next layer.

P: Previous layer.

C: Change echogram display Colormap (e.g., EK60, EK500, etc.)

Ctrl+S: Save current bottom/regions to XML file.

Ctr+Z: Undo last action (not available for all actions).

Ctr+Y: Redo last undone action (not available for all actions).

Home: Go to the start of an echogram (layer/transect)

End: Go to the end of an echogram (layer/transect)

Cursor modes:

0: "Normal" mode.

1: "Zoom In" mode.

Shift+1: "Zoom Out" mode.

2: "Bad Transmit" mode.

3: "Edit Bottom" mode.

4: "Create Region" mode.

5: "Measure Distance" mode.

6: "Draw Line" mode.

Mouse use in various modes:

i. In "Normal" mode:

Left-click on a region: Make region active.

M: Move user-created regions. Press M, click on a region (note how the cursor shape changes) and drag it.

Right-click on bottom/region/selection/display: Specific contextual menu. If on a region, this also makes it active.

Ctrl + left-click, drag & release: Create a rectangle selection box.

Shift + left-click, drag & release: Create a rectangle selection box spreading across all pings.

Double left-click: Delete selection box.


ii. In "Zoom In" mode:

Left-click* or *right-click: Zoom in.

Left-click, drag & release: Zoom-in box.

Right-click, drag & release* or *Ctrl + left-click, drag & release: Vertical zoom-in.


iii. In "Zoom Out" mode:

Left-click: Zoom out.

Right-click* or *Ctrl + left-click* or *double left-click: Zoom all the way out.


iv. In "Bad Transmit" mode:

Left-click: Flag ping as bad transmit.

Left-click, drag and release: Flag series of pings as bad transmits.

Right-click: Remove bad transmit flag.

Right-click, drag and release: Remove bad transmit flag for series of pings.


v. In "Edit Bottom" mode:

- "Classic Tool" is a mode combining hand-drawn and polyline to set the bottom manually:

Left-click: to start setting bottom, then...

a. Maintain left button pressed and drag: to create a hand-drawn section over a series of pings, or...

b. Release button and left-click again*: to create a polyline section over a series of pings.

c. Double left-click: to complete.

Right-click* or *Ctrl + left-click: to remove bottom for one ping.

Right-click, drag and release* or *Ctrl + left-click, drag and release*: to remove bottom for series of pings.


-"Spline Tool" is a mode allowing to "push" up or down an existing bottom for a series of ping.

Right-click: to open a contextual menu to set the size of the tool.

Left-click above the bottom and drag down: to push the bottom downwards.

Left-click below the bottom and drag up: to push the bottom upwards.


-"Brush Tool" is a mode allowing to remove obviously bad bottoms.

Left-click, drag and release: to define a rectangle area where bottom is to be removed.

Right-click, drag and release: to do the same AND also flag pings where a bottom was found as bad transmits


vi. In "Create Region" mode:

Left-click, drag & release: Create a region.

Left-click on a region: Make region active when in "Display Regions" (filled regions) mode.

Delete: Delete active region.


vii. In "Measure Distance" mode:

Left-click: to start a polyline, then..

a. Left-click: to add a vertex to the polyline. Finally...

b. Right-click: to delete the polyline.


viii. In "Draw Line" mode:

Right-click, keep pressed, drag and release: to create a hand-drawn line.

Shift + Left-click: to start a polyline, then...

a. Left-click: to add a vertex to the polyline. Finally...

b. Double left-click* or *Right-click: to complete the polyline.


Mini display tips & tricks:

The 'blue box' on the mini display represents the **extent of the main display**.

If the main display is currently zoomed-in, the blue box on the mini display can be **left-clicked and dragged around** and the main display will automatically update on release.

Likewise, if the main display is currently zoomed-in, a **left-click, drag and release** on another area of the mini display will create a new zoom-in blue box.

Right-click anywhere on the mini display to bring up a contextual menu allowing to undock the mini display.
