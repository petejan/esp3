Installation instructions

ESP3 currently only runs on PC with a 64bit version of Windows.

Currently you have two options to download and run ESP3, using either the MATLAB source code, or a compiled version.

- Option 1. Download and run the compiled version.

You will need:

i. To download and install the (free) Matlab Compiler Runtime R2017b(9.3) available at https://ch.mathworks.com/products/compiler/matlab-runtime.html.

Then:

1. Download the latest installer for the ESP3 compiled version on SourceForge available at https://sourceforge.net/projects/esp3/.

*Note: Earlier compiled versions are also available at https://sourceforge.net/projects/esp3/files/.

2. Run the installer and follow the instructions.



Option 2. Download the MATLAB source code, and run it within the MATLAB environment.

You will need:

i. MATLAB R2017b (or more recent).

ii. The following Licenced MATLAB toolboxes:

a. Signal Processing

b. Image Processing

c. Statistics and Machine Learning

d. Curve Fitting


iii. The following Installed MATLAB toolboxes:

a. Database (needs to be installed to have access to the sqlite functionnality of Matlab, but Matlab does not require a licence for the use of the sqlite embedded function...)


iv. Git available at https://git-scm.com/downloads and Git Large File Storage (Git LFS) available at https://git-lfs.github.com/.

* Note that instead of installing those stand-alone versions, you can simply download and use a Git client that will install them for you. We use and recommend 'Sourcetree' (https://www.sourcetreeapp.com/).

* If you don't know Git yet, learn about it and version controlling here .

Then:

1. Clone the ESP3 repository and check it out (download) on your machine.

2. Once you have the repository on your machine, open MATLAB, make the "ESP3" root folder your current directory, and launch "EchoAnalysis.m".

FAQs

- How do I stay up-to-date with the latest developments? Do I have to re-download a new version every time it comes out?

For the compiled version, yes. For the MATLAB version, Git will allow you to keep it updated to the latest version if you so wish.

- I would like to code my own MATLAB algorithms and extensions for ESP3. Can I do it?

Yes. Fork the project using Git and develop it on your side. Your changes will stay on your copy, without affecting the development of the main branch. You can always integrate the latest changes made on the main branch back into your forked copy (see Git functions "sync" and "merge").

If down the line you wish to suggest to integrate parts of your code to the main branch, you can do that too using "pull requests" in Git.

- I would like to help developing the main branch. Can I join the development team?

It would be a pleasure to have you onboard. Contact the team.
